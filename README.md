TainanBusMap
============

View Bus Routes of Tainan City

## Direct Link: 
http://kagami3421.github.io/TainanBusMap

## Instruction:

* Categoried Routes
* Visible/Invisible Bus Stops
* Change direction of route
* Hide/Show route-selected section
